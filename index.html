Welcome to the Theme-Dev Project

Theme-Dev is a WordPress theme framework built with Speed, Responsiveness and Search Engine Friendliness in mind.

Speed

Speed is optimized using a few different methods:

  1. Entirely imageless layout
  2. Lean, clean and optimized PHP
  3. Reducing database look-ups

Responsivemess

Theme-Dev has been built using a mobile first approach. That means that, while we take full advantage of the larger viewports, mobile is never compromised.

Javascript has been used to make a touch friendly pull menu to use for navigation on small screens.
